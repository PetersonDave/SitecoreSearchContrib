Help

Here are quick steps to follow if you are having trouble with getting searching to work properly.

1. Verify that your index is getting built

If your index is not getting updated, it's probably an issue with your Sitecore Search configuration, not the module.

2. Make sure that things are getting indexed

Use Luke. It's awesome.

3. Make sure your quieries are properly built.

3.1 Activate DEBUG mode

3.2 Use Luke to test the query
